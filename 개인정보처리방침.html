📱 커피프리 앱 개인정보처리방침

🔒 핵심 요약: 커피프리 앱은 어떠한 개인정보도 수집하지 않습니다.

📋 1. 개인정보처리방침 개요

커피프리('앱' 또는 '서비스')는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보처리방침은 커피프리 앱이 개인정보를 어떻게 처리하는지에 대해 설명합니다.

🚫 2. 개인정보 수집 및 이용

📌 중요: 개인정보 미수집

커피프리 앱은 다음과 같은 개인정보를 수집하지 않습니다:

  • 이름, 전화번호, 이메일 주소
  • 위치 정보
  • 기기 고유 식별자
  • 사용 패턴 및 행동 데이터
  • 기타 모든 개인 식별 정보

💾 3. 데이터 저장 및 처리

커피프리 앱은 다음과 같이 동작합니다:

  • 로컬 저장: 모든 설정 데이터는 사용자 기기 내부에만 저장됩니다
  • 서버 전송 없음: 어떠한 데이터도 외부 서버로 전송되지 않습니다
  • 오프라인 동작: 인터넷 연결 없이도 모든 기능이 정상 작동합니다
  • 데이터 암호화: 기기 내 설정 데이터는 안전하게 보호됩니다

🔐 4. 앱 권한 사용 목적

커피프리 앱이 요청하는 권한과 사용 목적:

  • 접근성 서비스: 자동 터치 기능 제공 (개인정보 수집 목적 아님)
  • 다른 앱 위에 표시: 오버레이 UI 표시 (개인정보 수집 목적 아님)
  • 정확한 알람: 자동 시작 기능 제공 (개인정보 수집 목적 아님)
  • 진동: 사용자 알림 제공 (개인정보 수집 목적 아님)

📊 5. 분석 및 광고

✅ 광고 없음: 커피프리 앱에는 광고가 포함되어 있지 않습니다.

✅ 분석 도구 없음: Google Analytics 등의 분석 도구를 사용하지 않습니다.

✅ 트래킹 없음: 사용자의 행동을 추적하지 않습니다.

🔄 6. 데이터 공유 및 제3자 제공

커피프리 앱은 수집하는 개인정보가 없으므로, 제3자와 데이터를 공유하지 않습니다. 앱은 완전히 독립적으로 작동하며 외부 서비스와 연동되지 않습니다.

👶 7. 아동 개인정보 보호

커피프리 앱은 만 13세 미만 아동을 포함한 모든 연령대에서 안전하게 사용할 수 있습니다. 어떠한 개인정보도 수집하지 않으므로 아동 개인정보 보호에 관한 우려가 없습니다.

🔒 8. 데이터 보안

비록 개인정보를 수집하지 않지만, 사용자 기기 내의 앱 설정 데이터는 Android 시스템의 보안 기능을 통해 안전하게 보호됩니다.

📝 9. 개인정보처리방침 변경

본 개인정보처리방침이 변경될 경우, 앱 업데이트를 통해 사용자에게 알려드립니다. 중대한 변경사항이 있을 경우 별도의 공지를 제공할 예정입니다.
